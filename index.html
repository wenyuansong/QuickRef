Quick Selenium Selector Reference

Written with StackEdit.

Goal XPath CssSelector
wild card By.XPath("//*[@class='error']") By.CssSelector(".error")
match text By.XPath("//button[contains(text(), 'Save')]") N/A
By Id By.XPath("//button[@id='main']") By.CssSelector("button#main")
By class By.XPath("//button[@class='main']") By.CssSelector("button.main")
by property By.XPath("//button[@name='Agree']") By.CssSelector("input[name='Agree']")
exclude(not) By.CssSelector(@"input[type='checkbox']:not([class='checkall'])") //find all checkbox without class=checkall
attribute contains By.CssSelector(@"input[name~='IsTelemarketingConsented']")
multiple condition By.CssSelector(@"input[name='A'][class='B']")
attribute with whitespace By.CssSelector(li.attendee.required) //li class="attendee required"
not adjective siblings li.attendee.required ~ li.options a // li.option is one of siblings of li.attendee.required
direct adjective siblings li.attendee.required + li.options a // li.option is followed directly after li.attendee.required
